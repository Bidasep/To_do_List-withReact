Nome do projeto :
TO-DO com React Avançado

Tecnologias utilizadas:
Html-CSS3 - JavaScript(ES6+) - Vite - React ( useEffect, useMemo, memo, useContext, useCallBack ), 
LocalStorage API, Context API, hooks customizados

Objetivo
Desenvolver uma aplicação de lista de tarefas (Todo List) utilizando recursos avançados do React. 
O foco será aplicar Hooks, Hooks customizados, Memoization e Context API, promovendo organização, reuso de lógica e performance.

Como rodar localmente o projeto:
Pré-requisitos

Antes de começar, você precisa ter instalado em sua máquina:

Node.js (versão 18 ou superior recomendada)
npm (geralmente já vem com o Node.js)
Git
Clone o repositório

Abra o terminal e execute:

git clone https://github.com/Bidasep/todo-react-avancado.git

Entre na pasta do projeto:

cd todo-react-avancado
Instale as dependências

Execute:

npm install
Inicie o servidor de desenvolvimento

Execute:

npm run dev
Abra no navegador

Após iniciar, o terminal mostrará um endereço parecido com:

http://localhost:5173/

Abra esse endereço no navegador para visualizar a aplicação.

Build para produção (opcional)

Para gerar a versão otimizada do projeto:

npm run build

Para visualizar a build localmente:

npm run preview
